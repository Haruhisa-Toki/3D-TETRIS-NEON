# 3D TETRIS — NEON

ブラウザで遊べるネオン×シンセウェーブ風の3Dテトリスです。
5×5の井戸にポリキューブを落とし、水平レイヤーを揃えて消していく Blockout スタイルのゲームです。

ビルド不要・依存インストール不要。`index.html` を開くだけで動きます。

## デモ

GitHub Pages を有効にすると以下のURLで遊べます。

```
https://<ユーザー名>.github.io/<リポジトリ名>/
```

## 遊び方

ローカルで遊ぶ場合は `index.html` をブラウザで開くだけです（Three.js をCDNから読み込むためネット接続が必要です）。

### ルール

- 5×5×高さ12の井戸にピースが落下します
- 水平レイヤー（5×5の1段）をすべて埋めると消去され、スコアが入ります
- 複数レイヤー同時消しでボーナス、3レイヤー消すごとにレベルアップして落下速度が上がります
- ピースが井戸からあふれるとゲームオーバーです

### 操作

| キー | 動作 |
| --- | --- |
| ← → ↑ ↓ | 水平移動 |
| Q / W / E | 回転（Y軸 / X軸 / Z軸） |
| C | ソフトドロップ |
| Space | ハードドロップ |
| P | 一時停止 |
| R | リスタート |
| マウスドラッグ | 視点回転 |
| ホイール | ズーム |

6秒間操作しないとカメラが自動でゆっくり旋回します。

## 特徴

- **ネオンビジュアル** — ブルームポストプロセスによる発光、シンセウェーブ風の地平線グリッドと星空
- **8種類のピース** — 平面ピース（I4・I3・L・T・S・O）に加え、立体コーナー・2段塔の3D専用ピース
- **ゴースト表示** — 着地位置を半透明でプレビュー
- **演出** — レイヤー消去時のパーティクル爆発・フラッシュ・画面シェイク、スコアポップアップ
- **サウンド** — WebAudio によるリアルタイム合成効果音（音声ファイル不要）
- **次ピースプレビュー** — 回転する3Dプレビュー表示

## 技術スタック

- [Three.js](https://threejs.org/) r160（CDN / importmap で読み込み）
- Vanilla JavaScript（ES Modules）・HTML・CSS のみ
- 単一ファイル構成（`index.html`）

## GitHub Pages での公開手順

1. このリポジトリの **Settings → Pages** を開く
2. **Source** を `Deploy from a branch`、ブランチを `main`（ルート `/`）に設定
3. 数分後に `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開されます
